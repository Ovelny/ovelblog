Network switching overview

Switches

  • Send or receive data frames
  • Keep a table of all MAC addresses
  • Avoid loops with the SPT protocol
  • Associate MAC addresses with the outgoing interface
  • Can be wired in series between themselves
  • Act as a relay between themselves
  • Records the sender of data frames

ARP

  • Address resolution protocol: finds the MAC address from an IP address
  • Send a broadcast message if the MAC address isn't found in cached table
