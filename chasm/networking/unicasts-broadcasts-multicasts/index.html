Unicasts, broadcasts, multicasts

Unicast

  • One station to one station: client to server for instance
  • Not suited for real-time communications

Broadcast

  • One station to all other stations
  • A package is received by everyone else
  • Scope: the broadcast domain
  • Used for ARP requests
  • Not used with IPv6

Multicast

  • One station to several (not all) other stations
  • A compromise between unicast and broadcast
  • Used for multimedia
  • Specialized and not suited for large networks
