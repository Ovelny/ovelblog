Cmus

Cmus is a lightweight, fast console music player for Unix-like operating systems.

Import music

Launch cmus and type :add with the path to your music, for instance: :add ~/Music. Cmus may take some time to import everything, depending of how many tracks you have.

Note that cmus will read, but never edit the metadata of your tracks.

Main commands

j/k - scroll down/up through the list of artists or tracks

tab - switch focus between list of artists / list of related tracks

enter - start playback

c- pause playback

v - stop playback

b - next track

z - previous track

s - toggle shuffle

m - toggle 'aaa' mode (artist, album, all). This defines the scope of shuffling for the next tracks. You can see the current mode in the bottom right of the interface.

x - restart track

i - jump focus to the track that is currently being played

/ - search anything (artists, tracks, albums...) in cmus

- - reduce volume by 10%

+ - increase volume by 10%
