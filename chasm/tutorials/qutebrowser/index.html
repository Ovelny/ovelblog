Qutebrowser

Qutebrowser is a web browser with a minimal GUI, licensed under the GPL. The navigation is keyboard-based and the key bindings are heavily inspired from vim.

Main shortcuts

j - scroll down

k - scroll up

h - scroll left

l - scroll right

gg - scroll to top

G - scroll to bottom

r - reload page

R - hard reload page (without cache)

J - go to next tab

K - go to previous tab

d - close current tab

co - close other tabs

u - restore most recently closed tab

H - backward in history

L - forward in history

f - enable hints, which are essentially key combinations to click on links

F - enable hints and open link in new tab

/ - search on page

n - search next occurrence

N - search previous occurrence

o - open a URL or a search query in current tab

O - open a URL or a search query in a new tab

yy - yank (copy) current URL

;y - enable hints to yank selected link

ctrl + p - pin tab

Main modes

i - insert mode, automatically activated when clicking any text field

v - caret mode, to navigate text. Press space to select text with h/j/k/l

: - command mode

esc - back to normal mode

Useful commands

:adblock-update - update and set the integrated, basic ad blocker

:home - go to home page

:history - show history

:history-clear - clear all browsing history

:config-clear - set all settings back to default

:q - exit qutebrowser

:wq - save current tabs and exit qutebrowser

Disable and control JavaScript

Use the command :set content.javascript.enabled false to globally disable JavaScript. You can then use the following commands to control exceptions:

tsh - enable JavaScript with the current host for this session

tSh - enable JavaScript with the current host permanently

tsH - enable JavaScript with the current host, subdomains included

tsu - enable JavaScript only for the current URL

Set qutebrowser as your default browser

On Ubuntu-based distros, simply run xdg-settings set default-web-browser org.qutebrowser.qutebrowser.desktop.

Change search engines

You can change the default search engine, and add other ones by binding them to shortcuts. The default setting is the following: {"DEFAULT": "https://duckduckgo.com/?q={}"}.

Change the DEFAULT value if you'd like to, or add some others search engines with another key/value pair.

Example: :set url.searchengines '{"d": "https://lite.duckduckgo.com/lite/?q={}", "DEFAULT": "https://www.google.com/search?hl=en&q={}"}'

This will use google as the default search engine, and duckduckgo lite whenever your search is prepended with d.

Play youtube videos with mpv

qutebrowser can't block youtube ads, but it can play youtube videos with mpv and youtube-dl for the same result. If you have mpv installed on your system, run this command in qutebrowser:

:bind ,f hint links spawn mpv --window-maximized=yes {hint-url}

Selecting a link with ,f will now hook the url to mpv in a maximized window, without any ads. It also works for live streams.
